# nodejs > modules

MODULOS DE CODIGO EN NodeJS + --------------------------- node funciona con MODULOS de código JS (cada modulo de código es un fichero con extension .js) cada modulo es una función inmediata de JS q realmente no ves, pero se genera para verla ejecutar en consola: node -e "const Module=require('node:module'); console.log(Module.wrapper);" una funcion inmediata en javascript es una funcion q se declara y ejecuta en el mismo momento q se define, tiene este formato: ( function (arg1, arg2, ...){ ...codigo funcion...} )(valor1,valor2,...)
entonces un modulo de codigo nodejs, cuando te creas dentro de un proyecto de node un fichero con extension .js, dicho fichero sin tu poner nada, se añade en la cabecera lo siguiente: ( function(exports, require, module, __filename, __dirname){ //...código js de tu modulo... de forma interna puedes usar: - exports: puntero al objeto module.exports (objeto donde se definen valores a exportar del modulo) - require: función q sirve para importar valores de otros modulos se usa asi: const variable=require('ruta_modulo' | 'nombre_modulo_nodejs') - module: objeto de la clase global Module de NodeJS (con sus props y métodos) - __filename: string, con ruta y nombre del modulo - __dirname: string con ruta del modulo } )(global) ------- \---> de los objetos globales de nodejs, se le pasan a la función inmediata del modulo los valores exports, require, module, __filename, __dirname
