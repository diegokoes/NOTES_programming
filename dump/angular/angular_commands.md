- Install Angular

```bash
		npm install -g @angular/cli
		ng version
```

- Create angular project

```bash
		ng new ebay-angular
```

- Run project

```bash
		ng serve [ -o | --open ] 
```

- Generate component:

```bash
ng generate component [nombre] --flat --skip-tests
		ng g c [nombre] --flat --skip-tests
```
